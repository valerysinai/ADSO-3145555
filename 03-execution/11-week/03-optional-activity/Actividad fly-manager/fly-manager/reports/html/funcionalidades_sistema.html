Contrato de implementacion

  • Modelar itinerario con `ticket_segment` y `flight_segment`.
  • Recrear baseline con `recrear_instalacion_limpia.ps1` antes de pruebas.
  • Validar cambios con `ejecutar_gate_pre_release.ps1`.
